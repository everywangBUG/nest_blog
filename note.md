# nest学习笔记
## 全局安装pnpm(硬连接)
* npm install -g pnpm

## 全局安装nest
* pnpm install -g @nestjs/cli nodemon ts-node

## 创建nest项目
* nest new project-name

## vscode安装Prisma插件

## 模块/控制器/路由/服务器概念
* 模块：一个模块就是一个文件夹，里面包含一个或多个控制器
* 控制器：一个控制器就是一个文件夹，里面包含一个或多个路由
* 路由：一个路由就是一个文件夹，里面包含一个或多个路由
* 服务器：一个服务器就是一个文件夹，里面包含一个或多个模块
